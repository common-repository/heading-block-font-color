== Description ==
This Plugin is made for setting the Font color for heading blocks which is not available by default for Headign block in Gutenberg.

== Installation ==

= Minimum Requirements =

* WordPress 4.9.9 or higher

= Manual Installation =

1. Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting older versions if they exist
2. Activate the plugin in your WordPress admin area.
